# 📏 Quantity Measurement App - Frontend

## 📌 Overview
This is the frontend of the **Quantity Measurement Application**, built using **HTML, CSS, and JavaScript**.  
It provides a simple interface to perform unit conversions and basic operations across different measurement types.

---

## 🚀 Features

- 🔄 Convert units (Feet, Inches, Kg, Gram, etc.)  
- ➕ Perform basic operations (Addition, Comparison)  
- 📏 Supports multiple measurement types:  
  - Length  
  - Weight  
  - Volume  
  - Temperature  
- 🎯 Clean and user-friendly interface  
- 🔗 Integrated with backend APIs  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript
